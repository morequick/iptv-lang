## 1. 语言包翻译流程
1. 翻译分割的语言文件
    ```
    例如: EN/01.txt
    ```
2. 以中文语言包为基础生成相应的语言包
    ```
    1> 合并分割的语言文件 => CN_EN.txt
    2> 以中文语言包为基础生成相应的语言包 => dist/EN.txt
    ```
